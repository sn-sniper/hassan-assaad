# 🚀 Hassan Assaad – 3D Portfolio Website

This is an interactive and animated developer portfolio built with modern web technologies like **React**, **TypeScript**, **Three.js**, **GSAP**, and **Vite**. The project showcases professional skills, projects, and a unique 3D user experience, designed to leave a lasting impression.

<!-- ![Preview Screenshot](./preview.png) -->

---

## ✨ Features

- 🎨 **3D Graphics** using [Three.js](https://threejs.org/) to bring your portfolio to life
- 🔥 **Smooth Animations** powered by [GSAP](https://greensock.com/gsap/)
- ⚡️ Blazing-fast development with [Vite](https://vitejs.dev/)
- 🧠 **TypeScript** for better type safety and scalability
- 💻 Fully responsive design for all devices
- 🧭 Interactive navigation and smooth scrolling
